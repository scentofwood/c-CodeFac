
컴파일러 위치
C:\Program Files (x86)\Dev-Cpp\MinGW64\bin\

======================================================

ㅁㅁ c 컴파일 ㅁㅁ

릴리즈용 컴파일 : gcc 소스파일.c -o 실행파일.exe
디버그용 컴파일 : gcc 소스파일.c -o 실행파일.exe -g3

======================================================

ㅁㅁ c++ 컴파일 ㅁㅁ

릴리즈용 컴파일 : g++ 소스파일.cpp -o 실행파일.exe
디버그용 컴파일 : g++ 소스파일.cpp -o 실행파일.exe -g3

링크 필요하다면 : g++ 소스파일.cpp -o 실행파일.exe -lws2_32
                  위 예제는 #include <winsock2.h> 쓰는 경우
